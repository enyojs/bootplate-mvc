Welcome

Welcome to the new MVC bootplate sample application. This application was designed to help show some of the new features being added in Enyo.js 2.2 as well as those provided by the new, optional mvc library. While we couldn't detail every line of code or feature added to the framework, this entire app is fully annotated and you can access the annotated source by selecting a file from the Annotated Source button on the menu above.

There are a few things to note when using this application. The top portion is fully interactive! It displays the use of enyo.Bindings in real-time. You can edit and add new records to be displayed in the Roller by clicking the buttons in the top menu. Have fun and play around. Also notice the Models and Showing labels in the top toolbar. These are also examples of using enyo.Bindings. When you're done playing have a look at the source.
